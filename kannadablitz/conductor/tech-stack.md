# Technology Stack

- **Programming Language:** TypeScript
- **Frontend Framework:** React (built with Vite)
- **Styling:** Custom CSS
- **Backend/Services:** Firebase (Hosting, Authentication, Firestore, Cloud Functions)
- **Key Libraries:** 
    - `canvas-confetti`: For celebration effects.
    - `html-to-image`: For generating shareable certificates/results.
    - `jimp`: For image processing (dev dependency).
