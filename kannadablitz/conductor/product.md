# Product Guide

## # Initial Concept
A React + TypeScript application built with Vite for learning Kannada.

## Target Audience
- Newcomers to Bangalore who want to learn survival Kannada.
- People who prefer a structured, short-term learning challenge.

## Core Features
- **Daily Challenges:** A 7-day or 30-day challenge structure to build a habit.
- **Interactive Scenarios:** Practice real-life conversations (e.g., auto driver, market).
- **Gamification:** Hearts system, streaks, and badges to motivate users.
- **Flashcards:** Vocabulary practice with flip cards.
- **Progress Tracking:** Dashboard to visualize daily progress and unlocked achievements.
- **Offline Support:** Full PWA support allowing for offline gameplay and automatic progress synchronization when back online.

## User Goals
- Learn essential Kannada phrases for daily survival in Bangalore.
- Build confidence in speaking and understanding basic Kannada.
- Complete a structured learning path with tangible milestones.

## Monetization (Optional)
- Free to use for the initial challenge.
- Potential for premium content or extended challenges in the future.
