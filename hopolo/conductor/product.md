# Product Guide: Hopolo

## Initial Concept
Build a production-ready commerce app that supports:
Clean storefront (homepage, PDP, cart)
Real-time cart using Firebase (Firestore)
Razorpay checkout
Order confirmation flow
User account management
Admin order + refund + shipping operations
Email notifications (order placed + delivery feedback)

## Target Audience
- **Individual Creators & Boutiques:** Small business owners who need a polished, visually appealing storefront that supports unique brand expression (e.g., emoji-friendly interfaces) without the complexity of enterprise tools.

## Core Goals
- **Frictionless Shopping Experience:** Deliver an "instant" feel with real-time cart syncing and seamless interactions to minimize friction and reduce cart abandonment.
- **Brand-First Design:** Enable a unique, emoji-centric feedback loop and clean aesthetic that helps small boutiques stand out.

## Key Features
- **Real-Time Cart:** Instant sync across devices and tabs using Firebase Firestore.
- **Clean Storefront:** Modern homepage with cinematic hero and best sellers, product detail pages (PDP), and cart interfaces.
- **Secure Checkout:** Integrated Razorpay checkout flow with webhook confirmation.
- **Operational Tools:** Admin dashboard with collapsible sidebar navigation for order management, refunds, shipping (Shadowfax), and dynamic storefront configuration (promo banner, hero section, reviews, maintenance mode).
- **Communication:** Automated email notifications for order confirmation and delivery feedback.

## Admin Panel Features
The Hopolo Admin Panel provides specialized tools for boutique owners to manage operations seamlessly:

- **📦 Inventory Management:** Add new products, track stock across variants (sizes/colors), and manage media assets for the storefront.
- **📊 Analytics Dashboard:** Monitor business performance with real-time sales metrics (Revenue, AOV, Conversion) and interactive trend charts.
- **🛍️ Storefront Configuration:** A "No-Code" interface to update the Cinematic Hero, toggle global promo banners, manage reviews, and enable maintenance mode.
- **🎫 Invitation System:** Securely onboard new staff (Admin, Manager, Editor) using phone-based authentication and single-use invite codes.
- **🧾 Order Management:** Hub for fulfillment including real-time order tracking, Shadowfax logistics integration, and processing refunds via Razorpay.
- **🏷️ Marketing & Promo Codes:** Create and manage discount codes (percentage or fixed) with purchase constraints and expiration dates.
- **✉️ Email Logs:** A transparent audit trail of all system-generated communication (Order Confirmations, Shipping Updates) with delivery status.
